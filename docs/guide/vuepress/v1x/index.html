              # VuePress（v1.x） 使用指南

              # 1. 创建并进入一个新目录

              mkdir vuepress-starter && cd vuepress-starter
              1

              # 2. 使用包管理器进行初始化

              yarn init # npm init
              1

              # 3. 将 VuePress 安装为本地依赖

              不再推荐全局安装 VuePress

              yarn add -D vuepress # npm install -D vuepress
              1

              注意

              如果你的现有项目依赖了 webpack 3.x，推荐使用 Yarn (opens new window)而不是 npm 来安装 VuePress。因为在这种情形下，npm 会生成错误的依赖树。

              # 4. 创建第一篇文档

              mkdir docs && echo '# Hello VuePress' > docs/README.md
              1

              # 5. 在 package.json 中添加一些 [scripts]

              {
  "scripts": {
    "docs:dev": "vuepress dev docs",
    "docs:build": "vuepress build docs"
  }
}
              1
              2
              3
              4
              5
              6

              # 6. 在本地启动服务器

              npm run docs:dev
              1

              VuePress 会在 http://localhost:8080 (opens new window)启动一个热重载的开发服务器。

              # 7.创建配置文件

              在 docs 目录下创建一个 .vuepress 目录，所有 VuePress 相关的文件都将会被放在这里。你的项目结构可能是这样：

              .
├─ docs
│  ├─ README.md
│  └─ .vuepress
│     └─ config.js
└─ package.json

              VuePress 网站必要的配置文件是 .vuepress/config.js，它导出一个 JavaScript 对象：

              module.exports = {
  title: 'Hello VuePress',
  description: 'Just playing around'

              # 1）配置导航栏logo

              可以通过 themeConfig.logo 增加导航栏 Logo ，Logo 可以被放置在公共文件目录（.vuepress/public/）：

              // .vuepress/config.js
module.exports = {
  themeConfig: {
    logo: '/assets/img/logo.png',
  }
}

              # 2）配置导航栏链接

              可以通过 themeConfig.nav 增加一些导航栏链接:

              // .vuepress/config.js
module.exports = {
  themeConfig: {
    nav: [
      { text: 'Home', link: '/' },
      { text: 'Guide', link: '/guide/' },
      { text: 'External', link: 'https://google.com' },
    ]
  }
}

              当提供了一个 items 数组而不是一个单一的 link 时，它将显示为一个 下拉列表 ：

              // .vuepress/config.js
module.exports = {
  themeConfig: {
    nav: [
      {
        text: 'Languages',
        ariaLabel: 'Language Menu',
        items: [
          { text: 'Chinese', link: '/language/chinese/' },
          { text: 'Japanese', link: '/language/japanese/' }

              # 3）配置自动生成侧栏

              可以通过配置在所有页面中启用它：

              // .vuepress/config.js
module.exports = {
  themeConfig: {
    sidebar: 'auto'
  }
}

              # 4）在文件中关闭侧边栏

              可以通过 YAML front matter 来禁用指定页面的侧边栏：

              ---
sidebar: false
---

              # 5）设置页面默认中文语言

              module.exports = {
    //设置页面默认语言
    locales: {
        // 键名是该语言所属的子路径
        // 作为特例，默认语言可以使用 '/' 作为其路径。
        '/': {
            lang: 'zh-CN', // 将会被设置为 <html> 的 lang 属性
        },
    },
}
