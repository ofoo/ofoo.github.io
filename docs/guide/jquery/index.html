    # jQuery 使用指南

    # 1. jQuery.each()方法

    实例

    输出每个 li 元素的文本：

    $("button").click(function(){
  $("li").each(function(){
    alert($(this).text())
  });
});
    1
    2

    定义和用法

    jQuery.each() 函数用于遍历指定的对象和数组。

    **提示：**返回 false 可用于及早停止循环。

    语法

    $(selector).each(function(index,element))
    1
    参数 描述
    function(index,element) 必需。为每个匹配元素规定运行的函数。
    index - 选择器的 index 位置
    element - 当前的元素（也可使用 "this" 选择器）
