      # MySQL 理论知识

      # 经验总结

      1. mysql 字符串查询是不区分大小写的。

      # 字段类型

      1. 小数类型：DECIMAL
      2. 整数类型：TINYINT（状态字段常用）、INT
      3. 日期类型：DATETIME
      4. 字符串类型：VARCHAR、TEXT（大文本常用）
